Macman

Instructions

Eat all of the pellets while avoiding the monsters! Eat fruit and powerups for extra points.

Controls

Utilize the arrow keys to move the Macman character.

About Macman

The Macman game was inspired by the classic Pacman arcade game and includes a Macalester College theme. The game was developed by three Macalester College students for their Internet Computing class project. More information about the developers can be found by clicking the link below:
